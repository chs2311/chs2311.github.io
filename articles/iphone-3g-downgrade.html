Downgrade iPhone 3G to iOS 2.0

This tutorial shows how to downgrade an iPhone 3G from iOS 4.2.1 to iOS 3.1.3 and iOS 2.0

1. Requirements:

* iPhone 3G which was originally shipped with iOS 2.0 (otherwise Restore will fail)
* Charging Cable with Data Transfer
* Linux-PC or Mac
* GitHub account for gh authentification
* gh installed

2. Legacy-iOS-Kit

To install Legacy iOS Kit via Terminal, please run:

gh repo clone https://github.com/LukeZGD/Legacy-iOS-Kit.git

You need gh installed and authenticated with a GitHub account.
If you don't have a GitHub account or gh installed you can download and extract it from below.

Open a Terminal and go into the Legacy-iOS-Kit directory.
If you installed it via gh, run:

cd Legacy-iOS-Kit

Then connect your iPhone to the PC and run

.\restore.sh

3. iOS 3.1.3 Downgrade

If your iPhone is already on iOS 3.1.3 or below you can skip this step,
but make sure that the iOS Installation is working properly.

In the Legacy-iOS-Kit Menu navigate to

Restore/Downgrade

iOS 3.1.3

Download Target

Start Restore

Follow the instructions shown in the Terminal.
Activate the Jailbreak, Enable the Memory Option for Custom IPSW and Disable Hacktivation
Get your Device into DFU/WTF-Mode. Instructions are shown in terminal.
Sometimes this might fail, you'll have to retry again.
